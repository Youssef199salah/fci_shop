project created with (React js and Laravel)
there are two projects one for front-end(React) and back-end(laravel)
there is a database file called back-end.sql
to run project you must have (xampp , phpmyAdmin , composer , node js , php )

1- open xampp and start (Apache and MySQL)
2- open MySQL and import database file, the database name should be (back-end)
3- open back-end project ,Go to .env file and change database connection to yours
4- open cmd in back-end project and run (php artisan serve)
5- open cmd in front-end project and run (npm install) then (npm strat)
